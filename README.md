# TASK
Task Manager

A simple C++ command-line application for managing personal tasks, with music playback using SFML.

Features
Add, view, edit, and mark tasks as completed
Persistent task list during runtime
Plays background music using SFML audio
User-friendly menu interface

Requirements
C++17 or later
SFML library (for audio playback)
Compatible .wav or .ogg music file (place in the project directory)

Usage
Compile the program with SFML linked.
Run the executable.
Follow the menu prompts to manage your tasks.

Notes
Music file path can be changed in the source code.
Tasks are stored in memory and are not saved after exit.
